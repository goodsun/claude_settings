# 複数のサブエージェントを使ってコードレビュー

## 実行手順
1. 変更されたファイルを確認
2. 以下の3つのサブエージェントを**並列実行**してレビューを実施
   - `senior-code-reviewer`: コード品質、可読性、保守性、バグ検出の観点からレビュー
   - `performance-tuning-expert`: パフォーマンス最適化の観点からレビュー
   - `api-design-reviewer`: API設計の観点からレビュー（APIコードの場合）
3. 各エージェントのレビュー結果を統合して報告
4. 重要度の高い指摘事項を優先的に提示

## 注意事項
- 必ず並列実行すること（単一メッセージで複数のTaskツールを呼び出す）
- 各エージェントの結果を待ってから統合レポートを作成
- API以外のコードの場合は`api-design-reviewer`をスキップ
- 日本語で結果を報告
