# PRを作成

- 以下のフォーマットに従うこと
- Draftで作成すること
- 宛先はstatingブランチにすること
- コマンドの例: `gh pr create --draft --title "title" --body "body"`
- DraftでPRを作成した後、open https://github.com/xxxxxx でDraftを開いてください。


■titleフォーマット
<ブランチ名のfeature/ag-xxxxのag-xxxxの部分をAG-xxxxにする> - <タイトル>

■bodyフォーマット
### asana

### ドキュメント

### 修正内容

### 確認内容（影響範囲、テスト内容など）

### 表示サンプル
